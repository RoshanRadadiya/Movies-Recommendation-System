# 🍿 Movie Recommendation System

A simple content-based movie recommendation system built with Streamlit that suggests similar movies based on your selection.

## 📥 Required Data Files

Due to GitHub file size limits, the data files are hosted on Google Drive.

| File | Download |
|------|----------|
| **movies.pickle** (Movie dataset with 4809 movies) | [![Download](https://img.shields.io/badge/Download-movies.pickle-blue?style=for-the-badge&logo=google-drive)](https://drive.google.com/file/d/1s0wP0iXJ92Mw36KNIV6Q1-zts4fkzqrA/view?usp=drive_link) |
| **similarity.joblib** (Pre-computed similarity matrix) | [![Download](https://img.shields.io/badge/Download-similarity.joblib-green?style=for-the-badge&logo=google-drive)](https://drive.google.com/file/d/1-xhrPzflTV6wzJixoxGEsA-2g3LZXI5M/view?usp=drive_link) |

## 🎯 Features

- Select any movie from dropdown list of 4809 movies
- Get 5 similar movie recommendations
- Simple and intuitive web interface
